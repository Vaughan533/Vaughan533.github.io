Links

External Links

I want these words to link to Google.

External links require an absolute URL, which is a reference using "http://". href="http://google.com" creates the hyperlink reference, target="_blank" opens the link in a new window, and title="Google" creates the hover title for greater accessability

Internal Links

I want to link to a page about me.

Internal links use a relative url, which references a file in the local directory. a href="about.html" references the about.html file in the folder

Mailto Links

I want an email to be sent to me.

Lists and Tables

Lists

  • Unordered lists (ul) use bullet points.
  • List terms are nested with li.
  • Kittens are delightful.
  1. Ordered lists (ol) will use numbers.
  2. Kittens are wonderful.
Definition List
A list using a involving a term (dt) and a definition (dd).

Tables

Tables are used for tabular data only, and should not be employed for page formatting.

Use the container tag table, and then use tr to create rows.

Within each tr tag, data is nested with td.
