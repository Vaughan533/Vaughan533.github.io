Links

External Links

I want these words to link to Google.

External links require an absolute URL, which is a reference using "http://". href="http://google.com" creates the hyperlink reference, target="_blank" opens the link in a new window, and title="Google" creates the hover title for greater accessability

Internal Links

I want to link to a page about me.

Internal links use a relative url, which references a file in the local directory. a href="about.html" references the about.html file in the folder
